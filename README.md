# Steps

1. Install python
2. Create virtual environment with python version 3.12

```bash
  python3.12 -m venv .venv
```

3. Activate venv

- Windows

```bash
.venv/Scripts/activate
```

- Linux with fish

```bash
source .venv/bin/activate.fish
```

4. Install requirements

```bash
pip install -r requirements.txt
```
